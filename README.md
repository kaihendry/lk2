# Install

	curl -sfL https://install.goreleaser.com/github.com/kaihendry/lk2.sh | sh

# Via Docker

Binds to port 9000 by default

	docker run -p 9000:9000 -v $YOUR_MEDIA_DIR:/media hendry/lk2 /lk2 /media
